# @hexajs-dev/cli

CLI and build compiler for HexaJS browser extension projects.

## Install

```bash
npm install -g @hexajs-dev/cli
```

## Quick Start

```bash
hexa new my-extension
cd my-extension
pnpm install
hexa build --platform chrome
```
