# @hexajs-dev/ports

Platform ports and browser abstraction contracts for HexaJS.

This package contains the cross-context messaging and platform-facing contracts used by background, content, and UI contexts.

## Install

```bash
npm install @hexajs-dev/ports
```

## Package Exports

- `@hexajs-dev/ports`
- `@hexajs-dev/ports/hexa-metadata.json`
- `@hexajs-dev/ports/hexa.web.ext`
