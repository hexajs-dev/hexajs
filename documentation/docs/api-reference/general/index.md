# General Ports

General ports are available in any runtime context — background, UI, or content — with no context restrictions.

## Included Ports

| Port | Description |
|---|---|
| [RuntimePort](./runtime-port.md) | Cross-context messaging, lifecycle hooks, and extension reload |
| [I18nPort](./i18n-port.md) | Localization string resolution and language detection |
| [ExtensionPort](./extension-port.md) | Resource URL resolution and extension view access |
